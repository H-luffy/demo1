# 课表模板编辑系统 - 使用说明

## 📋 项目概述

这是一个完整的课表模板编辑系统，支持：
- 管理员上传模板图片并手动标记格子位置
- 用户选择模板并在格子中编辑内容
- 导出最终课表为 PNG 图片

## 🚀 快速开始

### 1. 启动后端服务器

```bash
cd backend
npm install
npm start
```

服务器将在 http://localhost:3001 运行

### 2. 启动前端开发服务器

```bash
cd frontend
npm install
npm start
```

应用将在 http://localhost:3000 运行

## 📝 功能使用指南

### 管理员操作

#### 1. 上传模板

1. 访问 http://localhost:3000
2. 点击导航栏的"模板管理"
3. 点击"上传新模板"按钮
4. 选择课表模板图片（建议使用清晰的课表图片）
5. 图片上传后会自动显示

#### 2. 标记格子

1. 点击"标记模式"按钮
2. 在下拉菜单中选择星期（周一到周五）
3. 设置节次索引（0-7，例如第1节课就是0，第2节课就是1）
4. 将鼠标移动到图片上，光标会变成十字形
5. 在要标记的格子位置按下鼠标左键
6. 拖动鼠标创建矩形区域
7. 松开鼠标完成标记
8. 格子会显示在图片上，并标注 day-index
9. 重复步骤 2-7，标记所有需要的格子

#### 3. 保存配置

1. 标记完成后，点击"保存配置"按钮
2. 配置会保存到服务器

### 用户操作

#### 1. 选择模板

1. 访问 http://localhost:3000
2. 默认会显示"模板列表"页面
3. 找到你想要编辑的模板
4. 点击模板卡片上的"编辑课表"按钮

#### 2. 编辑课表

1. 在格子中直接输入课程内容
2. 输入框背景透明，文字居中显示
3. 可以随时修改内容

#### 3. 导出图片

1. 编辑完成后，点击"导出图片"按钮
2. 系统会自动下载 PNG 图片
3. 图片包含完整的模板背景和课程内容

## 🔧 技术特性

### 图片处理

- 所有图片都使用 Base64 编码加载，避免跨域问题
- 导出时使用 html2canvas 库生成图片
- 支持高分辨率导出（scale: 2）

### 样式特性

- 输入框背景完全透明
- 文字居中显示
- 支持自定义字体和颜色
- 导出时保持原始样式

### 数据存储

- 图片：backend/uploads/
- 模板数据：backend/templates-data/

## 💡 使用提示

### 标记格子时

- 建议放大浏览器页面以获得更好的精度
- 如果标记错了，可以点击格子右上角的"×"按钮删除
- 可以点击"清除格子"按钮清除所有格子重新标记
- 标记时建议使用较大的图片以获得更好的精度

### 编辑课表时

- 输入框背景透明，可以清楚看到模板背景
- 文字自动居中显示
- 支持实时编辑，输入内容会立即显示

### 导出图片时

- 导出过程可能需要几秒钟，请耐心等待
- 导出的图片会自动下载到浏览器的默认下载目录
- 图片质量设置为最高（1.0）
- 支持高分辨率导出

## 🐛 常见问题

### Q: 导出的图片是白板怎么办？

A: 确保以下几点：
1. 后端服务器正在运行（http://localhost:3001）
2. 前端开发服务器正在运行（http://localhost:3000）
3. 模板图片已正确上传
4. 格子已正确标记并保存

### Q: 图片加载失败怎么办？

A: 检查以下几点：
1. 后端服务器是否正常运行
2. 上传目录（backend/uploads）是否有权限
3. 浏览器控制台是否有错误信息

### Q: 导出功能不工作怎么办？

A: 尝试以下步骤：
1. 刷新页面
2. 重新选择模板
3. 检查浏览器控制台是否有错误信息
4. 确保使用的是现代浏览器（Chrome、Firefox、Edge 等）

## 📞 技术支持

如果遇到问题，请检查：
1. 浏览器控制台是否有错误信息
2. 后端服务器日志是否有错误信息
3. 网络请求是否正常

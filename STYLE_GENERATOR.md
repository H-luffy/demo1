# 课表风格生成器使用说明

## 功能介绍
课表风格生成器允许用户上传一张课表图片，并选择想要渲染的风格（如科幻风、古风等），系统会调用千问AI生成用户想要的风格课表图片。

## 使用步骤

### 1. 环境配置

#### 后端配置
1. 在后端目录安装依赖：
```bash
cd backend
npm install
```

2. 配置千问AI的API Key：
   - 在环境变量中设置 `QWEN_API_KEY`
   - 或在 `.env` 文件中添加：
     ```
     QWEN_API_KEY=your_api_key_here
     ```

3. 启动后端服务器：
```bash
npm start
```

#### 前端配置
1. 在前端目录安装依赖：
```bash
cd frontend
npm install
```

2. 启动前端应用：
```bash
npm start
```

### 2. 使用功能

1. 在浏览器中打开 `http://localhost:3000`
2. 点击导航栏中的"风格生成器"
3. 点击上传区域选择一张课表图片
4. 在下拉菜单中选择想要生成的风格（科幻风、古风、卡通风格等）
5. 点击"生成风格化课表"按钮
6. 等待生成完成后，可以预览生成的图片
7. 点击"下载图片"按钮保存结果

## 支持的风格
- 科幻风
- 古风
- 卡通风格
- 水彩画风格
- 油画风格
- 像素艺术风格
- 赛博朋克风格
- 日式动漫风格

## 注意事项
1. 请确保已正确配置千问AI的API Key
2. 上传的图片建议为清晰的课表图片
3. 生成过程可能需要几秒钟到几分钟不等，请耐心等待
4. 生成的图片会保存在服务器的 `uploads` 目录中

## 技术栈
- 前端：React + Tailwind CSS + Lucide Icons
- 后端：Node.js + Express
- AI服务：阿里云千问AI (DashScope)
